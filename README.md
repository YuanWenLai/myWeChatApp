
## 音乐书籍小程序(前端)

[小程序服务端](https://github.com/YuanWenLai/wc_server)



## <font color=#00ffff>具体功能</font>
1. 音乐播放，点赞，分享，评论
2. 周刊展示，跳转，点赞
3. 书籍展示，书籍搜索，书籍查看及书籍评论点赞 
4. 个人收藏页面展示
